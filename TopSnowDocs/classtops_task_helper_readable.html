% readEvent

Useful utility for checking for an event and, if found, saving the (synchronized) timing of the event in the trial data struture. Arguments: acceptedEvents ... cell array of strings acceptedEvents to list names of events that can be used. task ... the topsTreeNode task caller eventTag ... string used to store timing information in trial struct. Assumes that the current trialData struct has an entry called time_<eventTag>.
